# HuaHuaDailyMCP

🌸 花花日记助手 - 让 OpenClaw 连通你的基金数据。

## 功能

- 查询基金持仓、今日收益、累计收益
- 管理交易记录
- 获取市场行情
- 向 App 发送交易请求信号

## 安装

```bash
# 方法一：从 ClawHub 安装（推荐）
clawhub install huahua-daily

# 方法二：从 GitHub 直接克隆
git clone https://github.com/baiye1997/HuaHuaDailyMCP.git
```

## 配置

在 OpenClaw 配置文件中添加环境变量：

```
BAIYE_AGENT_TOKEN=你的令牌
```

## 获取 Agent Token

1. 打开花花日记 App 端/网页端 → 设置页
2. 点击「Agent 访问令牌」
3. 点击「生成新令牌」，完成邮箱验证
4. 复制一次性显示的令牌
